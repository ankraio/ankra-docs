## ankra tokens

Commands to list, create, revoke, and delete API tokens.

## ankra tokens create

Create a new API token

```bash
ankra tokens create <name> [flags]
```

**Flags**

| Flag | Default | Description |
|------|---------|-------------|
| `--expires` |   | Token expiration date (ISO 8601 format) |
| `-o`, `--output` |   | Output format: json or yaml (default: human-readable) |
| `--scopes` | `[]` | Permission allowlist for the token (e.g. clusters.read,stacks.deploy); omitted grants the user's full authority |

## ankra tokens delete

Delete an API token. The token must be revoked first.

```bash
ankra tokens delete <token_id>
```

## ankra tokens list

List all API tokens

```bash
ankra tokens list [flags]
```

**Flags**

| Flag | Default | Description |
|------|---------|-------------|
| `-o`, `--output` |   | Output format: json or yaml (default: human-readable) |

## ankra tokens revoke

Revoke an API token (can be deleted after)

```bash
ankra tokens revoke <token_id>
```
